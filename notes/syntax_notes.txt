C++ SYNTAX

Example: Hello World
#include <iostream>
using namespace std;

int main() {
	cout << "Hello World!";
	return 0;
}

LOGIC:
Line 1: #include <iostream> is a header file library that allows the input and output of objects, like cout. Adds fuctionality to C++ programs.

Line 2: using namespace std; means that we can use names for objects and variables from the standard library.

Line 3: Blank lines are ignored by C++, but are used to make code easier to read.

Line 4: int main() is a funtion. any code inside its curly brakets {} will be executed.

Line 5: cout (pronounced see-out) is an object used together with the insertion operator << to output/print text. In the example it will print out Hello World!

NOTE: C++ is case sensitive. "cout" and "Cout" have different meanings.
NOTE: Every statement in C++ end with a semicolon ;
NOTE: the body of int main() can also be writen as: int main() { cout << "Hello World! "; return 0; }

Line 6: return 0; ends the main function
Line 7: Close curly bracket } to actually end the main function.
